# 論理回路図作成ツール (Logic Gate Editor)

Webブラウザ上で動作する、HTML/JavaScript/CSSベースの論理回路図作成アプリケーションです。ANDやORなどの基本的な論理素子を備え、ドラッグ＆ドロップやショートカットキーを用いた直感的な操作で回路図を描画できます。

## 🎯 目的
特別なソフトウェアのインストールを必要とせず、ブラウザだけで直感的かつ効率的に論理回路図を作成できる環境を提供すること。

## 🚀 主な機能
* **論理素子の配置:** AND, OR, NOT, NAND, NOR, XOR
* **接続用パーツの配置:** 黒丸, 白丸, 接続部拡張
* **自由な配線機能:** マウスドラッグによる配線の作成、および選択時の端点（白い丸）による位置調整
* **直交配線サポート:** `Shift`キーで水平、`Ctrl`キーで垂直な線を補正描画
* **テキスト追加:** テキストボックスを追加し、キャンバス上で直接インライン編集が可能
* **要素の操作:** 選択、移動、リサイズ（右下の青い丸）、複製（`Space`キー）、削除（`Delete` / `Backspace`キー）
* **ヘルプ機能:** ツールバーの「使用方法」ボタンから操作マニュアルをいつでも確認可能

## 💻 動作環境（使用プラットフォーム）
* **対象環境:** Webブラウザ (Google Chrome, Microsoft Edge, Safari, Firefox などのモダンブラウザ)
* ※マウス操作やキーボードショートカットを多用するため、**PC環境**での利用を想定しています。

## 🛠️ 開発環境
* **コードエディタ:** Visual Studio Code (VS Code)
* **バージョン管理:** GitHub, GitHub Desktop
* **開発支援AI:** Gemini (コード生成・設計アドバイス)
* **使用技術・言語:** HTML5, CSS3, JavaScript (Vanilla JS / フレームワーク不使用)

## 📝 使い方
1. Webブラウザを開き、次のURLにアクセスします。
   **👉 https://airdaaaaaa.github.io/LogicCircuitTool/**
2. 画面上部のツールバーから配置したい素子を選択し、キャンバス上で自由に回路図を作成してください。
3. 詳しい操作方法は、画面右上にある「使用方法」ボタンを押すといつでも確認できます。

## ⚠️ 非目標（本ツールで対応しないこと）
* 回路シミュレーション機能（入力に応じた出力の計算など）
* 真理値表の自動生成
* FPGAやマイコンへの書き込み機能
